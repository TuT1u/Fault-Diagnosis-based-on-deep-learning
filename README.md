# Fault-Diagnosis-based-on-deep-learning
## 基于深度学习的旋转机械故障诊断的实现   tensorflow和pytorch版本
本储存库提供基于深度学习的旋转机械故障诊断的一般框架，包括数据预处理、模型搭建、模型训练，供 **该领域的初学者**进行学习研究。
代码有 tensorflow 和 pytorch 两种深度学习框架的版本
